Программа для создания наклеек.
Версия 3.3.2
	Bugfix

Структура каталогов:

	stickers 3.3.2|
		    |
		    --export 		в этот каталог создаются каталоги с Excel-файлами для программы распила    
		    |
		    |
		    --template		в данном каталоге содержится Word-документ с шаблоном наклейки
		    |
		    |
		    --lib		системный каталог с библиотекми, необходимыми для работы приложения



В случае возникновения ошибки "Приложению не хватает памяти для обработки файла" необходимо закрыть приложение и запустить файл "start.bat".
