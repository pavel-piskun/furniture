Программа для заполнения шаблона данными из БД (шаблон и БД хранятся в Excel).

Версия 3.5
	- Изменено расположение таблиц с описанием материалов в базе шкафчиков
	- Изменено расположение таблиц в шаблоне
	
Структура каталогов:

   XLSFiller 3.5|
				|
				--config 	каталог с конфигурацией программы.
				|
				|
				--data		(Опционально) каталог для хранения БД.
				|
				|
				--lib		системный каталог с библиотекми, необходимыми для работы программы.
